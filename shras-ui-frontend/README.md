Frontend

Tech Stack: React, Tailwind CSS, Lucide Icons, Axios

Features
Interactive dashboard displaying students, rooms, and allotments.
Real-time updates for room availability.
Responsive UI for desktop and mobile devices.
Hoverable error descriptions for API responses.
Easy navigation with Navbar and structured layout.


# Clone the repository
git clone https://github.com/Jitender712/smart-hostel-room-allotment-system.git

# Navigate to frontend directory
cd smart-hostel-room-allotment-system/frontend

# Install dependencies
npm install

# Start development server
npm run dev


Usage
Open your browser at http://localhost:3000
Login or register (if authentication is implemented)
Navigate through dashboard to view students, rooms, and allotments
Hover over error indicators to see details of API failures
